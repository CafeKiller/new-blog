本主题已预装并配置了 [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) 集成，配置文件位于 `astro.config.ts`。如果你不想使用 MDX，可以从配置文件中移除该集成来禁用它。

## 为什么使用 MDX？

MDX 是一种特殊的 Markdown 格式，支持嵌入 JavaScript 和 JSX 语法。这使你能够[在 Markdown 内容中混合使用 JavaScript 和 UI 组件](https://docs.astro.build/en/guides/markdown-content/#mdx-features)，实现交互式图表或提示框等功能。

如果你已有 MDX 格式的内容，这个集成将帮助你轻松迁移到 Astro。

## 示例

下面展示如何在 MDX 中导入和使用 UI 组件。
在浏览器中打开此页面时，你应该能看到下方的可点击按钮。

import { Button } from 'astro-pure/user'

<Button >点击我</Button>
