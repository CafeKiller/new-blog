import { Aside, Tabs, TabItem, MdxRepl } from 'astro-pure/user'

组件让你可以轻松地一致地重用 UI 或样式。你不仅可以在 `.astro` 文件中使用它们，还可以在 `.mdx` 文件中使用。

<Aside type='tip'>
[MDX](https://mdxjs.com/) 是一种允许你在 Markdown 中嵌入 JSX 的格式。在其他方面与 markdown 文件没有区别。
</Aside>

对于 `.astro`，你可以直接导入和使用组件。第一节也会展示一个示例。

<Aside type='tip'>
如果你总是在内容中导入某些组件，可以使用 [astro-auto-import](https://github.com/delucis/astro-auto-import/tree/main/packages/astro-auto-import) 自动导入它们，而不是在每个文件中手动导入。
</Aside>
