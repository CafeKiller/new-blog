## 移除友链页面

友链页面用于展示你的友情链接，默认启用。

如果你想移除友链页面，请按照本节步骤操作。否则，可以跳到下一章节。

在 `src/site.config.ts` 中将 `links.enable` 设置为 `false`：

```ts title="src/site.config.ts"
export const integ: IntegrationUserConfig = {
  // ...
  links: {
    enable: false // [!code highlight]
  }
}
```

同时，记得删除以下文件夹和文件：

- `src/components/links/`
- `src/pages/links/`
- `public/links.json`

## 基本配置

友链的基本配置在 `src/site.config.ts` 中。你可以添加友链日志或你自己的链接信息。

```ts title="src/site.config.ts"
export const integ: IntegrationUserConfig = {
  links: {
    // 友链日志
    logbook: [
      { date: '2025-03-16', content: '有漏水吗？' },
      { date: '2025-03-16', content: '什么漏水？' },
      { date: '2025-03-16', content: '我这里全是水，到处都是水！' },
      { date: '2025-03-16', content: '肯定是水管的问题。' },
      { date: '2025-03-16', content: '让我们把这句话加入名言录。' },
    ],
    // 你自己的链接信息
    applyTip: {
      name: theme.title,
      desc: theme.description || '无',
      url: 'https://astro-pure.js.org/',
      avatar: 'https://astro-pure.js.org/favicon/favicon.ico'
    }
  },
}
```

## 集成 `Friend-Circle-Lite`

[Friend-Circle-Lite](https://github.com/willow-god/Friend-Circle-Lite) 是一个精简的友链应用，无需后端，仅使用 GitHub Action 运行。

它需要：

1. 一个启用了定时 GitHub Actions 的 GitHub 仓库。
2. 一个静态站点服务器，如 Vercel、Netlify、GitHub Pages 等。

本主题未集成它，将来也不会提供支持。但别担心，本文档将指导你进行集成。

<Steps>

1. Fork [Friend-Circle-Lite](https://github.com/cworld1/friend-circle-lite)。
2. 修改你 fork 仓库中的 `config.yaml`：

   ```yaml title="config.yaml"
   spider_settings:
      enable: true
      json_url: "<你的站点>/links.json"
      article_count: 4
   ```

3. 进入"Actions"页面，手动运行"Friend Circle Lite"工作流以检查是否正常工作。这也会在"page"分支生成服务器文件。

4. 查看[文档](https://github.com/willow-god/Friend-Circle-Lite?tab=readme-ov-file#%E9%83%A8%E7%BD%B2%E9%9D%99%E6%80%81%E7%BD%91%E7%AB%99)将服务器文件部署到你的静态站点服务器。

5. 将[获取脚本文件 `friendCircle.ts`](https://gist.github.com/cworld1/cf0dc6daa89264e6d745d3e68ef8c01c) 添加到项目的 `src/plugins` 路径。

6. 将[样式文件 `fc.css`](https://gist.github.com/cworld1/cf0dc6daa89264e6d745d3e68ef8c01c?permalink_comment_id=5311916#gistcomment-5311916) 添加到项目的 `src/assets/styles` 路径。

7. 在 `src/components/pages/links/index.astro` 中添加初始化代码。

</Steps>
