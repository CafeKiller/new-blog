## 贡献代码

非常感谢所有为 [astro-theme-pure](https://github.com/cworld1/astro-theme-pure) 做出贡献的人。开源社区因你们而精彩。

[![contributors](https://contrib.rocks/image?repo=cworld1/astro-theme-pure)](https://github.com/cworld1/astro-theme-pure/graphs/contributors)

<Aside title='贡献指南'>
贡献错误修复或新功能是最直接的贡献形式。最好先创建一个 issue 来讨论你想要更改的内容，这可以避免不必要的工作，特别是当你想进行破坏性更改时。

对 `astro-pure` 包的所有更改都应该先启用[自定义主题#包模式](/docs/advanced/customize#package-mode)。PR 测试也将在该模式下运行。这要求你首先使用 `bun run yijiansilian` 命令在本地测试你的更改。

对于贡献文档，你可以简单地点击每个文档页面底部的"在 GitHub 上编辑"链接来更改文档内容。
</Aside>
